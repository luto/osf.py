# osf.py
Parse OSF in Python.

## Install
```
$ pip install openshownotesformat
```

## Setup

Use python3.

```
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ pip install -e .
```

## Tests
```
$ py.test --cov-config .coveragerc --cov osf --cov-report html test
```
